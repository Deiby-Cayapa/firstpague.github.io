Estructura Basica de HTML

Doctype

Nos indica el tipo de documento con el que estamos trabajando, en este caso HTML.



Html Lang

Esta etiqueta nos sirve para definir el lenguaje natural con el que vamos a trabajar, por ejemplo: "en" define que trabajaremos en ingles.



Head

Es el encabezado del documento html, donde ubicamos aquello que el usuario no verá directamente pero que es escencial para que funcione el sitio web.



Metaetiquetas

Son elementos que se incorporan en el encabezado y que resultan invisibles para el visitante normal, pero de gran utilidad para navegadores, diseñadores y programadores.

ATRIBUTO DESCRIPCION SINTAXIS
Description Descripción del contenido
Keywords Pueden usarlas para que el motor de busqueda tenga mas informacion acerca del contenido de la pagina.
Charset Indica la condificacion de caracteres.

Title

Esta etiqueta nos sirve para darle un título general a la página que es visible para el usuario, se ubica en la pestaña del navegador.



Link

Especifica la relacion entre el documento HTML y un recurso externo.



Body

Se refiere al cuerpo del documento, es lo que el usuario va poder visualizar.
